# Italiaanse leerapp

Eerste publiceerbare webapp-versie van de Italiaanse leerapp.

## Lokaal starten

1. Open Terminal in deze map
2. Installeer dependencies:

```bash
npm install
```

3. Start de app:

```bash
npm run dev
```

4. Open de lokale link die Vite geeft, meestal:

```bash
http://localhost:5173
```

## Publiceren via GitHub en Vercel

### GitHub
1. Maak in GitHub een nieuwe repository aan, bijvoorbeeld `italiaanse-leerapp`
2. Upload alle bestanden uit deze map in die repository

### Vercel
1. Log in bij Vercel
2. Klik op **Add New** > **Project**
3. Kies je GitHub-repository
4. Vercel herkent automatisch dat dit een Vite-project is
5. Klik op **Deploy**

## Op iPhone zetten

1. Open de Vercel-link in Safari
2. Tik op de deelknop
3. Kies **Zet op beginscherm**
